# Building Pong with Love2D & Lua

Step by step instructions on how to build pong with Love2D and Lua.

## Run
```bash
git clone git@github.com:CoderDojoRotterdam/love2d-pong.git
cd love2d-pong
npm install
npm start
open love2d_pong.pdf
```
